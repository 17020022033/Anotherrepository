# 172. 阶乘后的零
## 题目
给定一个整数 n，返回 n! 结果尾数中零的数量。  

示例 1:  

输入: 3  
输出: 0  
解释: 3! = 6, 尾数中没有零。  
示例 2:  

输入: 5  
输出: 1  
解释: 5! = 120, 尾数中有 1 个零.  
说明: 你算法的时间复杂度应为 O(log n) 。  
## 思路
只有2和5乘起来才能乘出尾数0。但是2的数量比5要多，所以制约0的数量的其实是5的个数。 
所以要阶乘的数里面5的数量，就可以求出结果啦。
## 代码实现
```ruby
class Solution {
public:
    int trailingZeroes(int n) {
        int res = 0;
        while(n = n / 5)
        {
            res += n;
        }
        return res;
    }
};
```
